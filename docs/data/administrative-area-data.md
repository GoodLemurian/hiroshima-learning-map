## 属性と利用方法

| 内容 | 属性 |
| --- | --- |
| 都道府県名 | `N03_001` |
| 市名 | `N03_004` |
| 区名 | `N03_005` |
| 地方公共団体コード | `N03_007` |
| 将来の統計CSVとの結合キー | GeoJSONの`N03_007`とCSVの`ward_code` |

`N03_007`は区単位のキーとして使用します。同一区コードを持つFeatureが複数存在するため、Feature単位の一意IDとしては使用しません。元データをdissolveせず、区の選択時はMapLibreのフィルターで同じコードを持つ全Featureを対象にします。

確認した区コードは `34101`、`34102`、`34103`、`34104`、`34105`、`34106`、`34107`、`34108` です。
